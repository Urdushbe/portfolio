Professional Project

ALBASENG.COM

🚀 Albaseng.com – Simplifying Job Search
The purpose behind creating Albaseng.com was to make job searching easier for foreigners in Korea. Until now, people looking for part-time jobs relied on various Telegram groups, where job seekers and employers posted manually. This was inefficient and unorganized.
🔹 With Albaseng.com, I aimed to bring all these users onto one platform.
Now, employers can easily post jobs, and job seekers can quickly find opportunities based on their location. Additionally, the platform includes a general chat section, allowing users to discuss issues, share experiences, and help each other.
Even though I have the skills to create a visually stunning frontend, I intentionally kept the UI simple and user-friendly to ensure that anyone can use the platform with ease.
⚠️ Current Status: The Platform is Closed
Unfortunately, Albaseng.com is no longer active due to a limited user base and insufficient funding for marketing.
Despite the technical success of the platform, reaching the right audience and sustaining growth required more investment in outreach and advertising, which was beyond my budget at the time.
However, this project was a valuable experience in building a scalable web application, understanding user needs, and learning about the challenges of launching a product.
💡 I remain passionate about solving real-world problems through technology, and I look forward to building new, impactful solutions in the future! 🚀

Bussness

🛠 Telegram Kino Bot – Automated Movie Delivery

This bot was designed to seamlessly deliver movies to users through Telegram, using a unique movie code system. It was built to integrate with Instagram reels marketing, where movie previews are shared, and users can obtain the full movie by entering a code into the bot.

💡 How It Works
📌 Step 1: A movie reel is posted on Instagram, featuring a unique movie code on the reel overlay.
📌 Step 2: The user sends the movie code to the Telegram bot.
📌 Step 3: The bot finds the corresponding movie and forwards it from a private Telegram channel to the user.
📌 Step 4: If a movie file is larger than 50MB (Telegram's limit for direct sending), the bot instead forwards it from the private channel, ensuring smooth delivery.

🔧 Technologies Used

* Aiogram Framework

* Telegram Bot API

* Python
